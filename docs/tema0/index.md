# 🔁 Tema 0: Introducción — Git, GitHub y Docker

!!! note "Tema de introducción — sin RA asociado"
    Este tema no evalúa ningún resultado de aprendizaje del currículo de Acceso a Datos. Repasa y sienta las bases de dos herramientas que vas a usar constantemente durante el curso: el control de versiones con Git y GitHub (si ya cursaste Entornos de Desarrollo, esto es un repaso) y una introducción a Docker, que usaremos más adelante para levantar bases de datos en contenedores sin instalarlas a pelo en tu máquina.

Antes de meternos con ficheros, bases de datos y ORMs, conviene tener controladas dos herramientas que van a acompañarte en cada actividad del módulo: vas a subir tu código a GitHub constantemente, y vas a necesitar levantar contenedores con distintos motores de bases de datos (MySQL, PostgreSQL, MongoDB…) sin volverte loco instalándolos uno a uno.
