# 📚 Acceso a Datos — 0486

Módulo de **Desarrollo de Aplicaciones Multiplataforma (DAM)**

En este módulo aprendes a que tus programas guarden y recuperen información de verdad: desde ficheros de texto o binarios hasta bases de datos relacionales, orientadas a objetos y documentales, pasando por herramientas ORM que te libran de escribir SQL a mano.

---

## 🎯 Qué vas a aprender

- 🗄️ Acceso a bases de datos relacionales mediante conectores: conexiones, consultas, procedimientos almacenados y transacciones.
- 🧬 Persistencia con herramientas ORM: mapear clases a tablas y trabajar con objetos persistentes en lugar de sentencias SQL sueltas.
- 🧱 Bases de datos objeto-relacionales y orientadas a objetos: cuándo tiene sentido usarlas y cómo acceder a ellas.
- 📄 Bases de datos documentales nativas (NoSQL): colecciones, documentos y sus lenguajes de consulta.
- 🧩 Componentes de acceso a datos reutilizables e integrables en cualquier aplicación.
